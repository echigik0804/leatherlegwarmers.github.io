# Leather Leg Warmers Website

This repository contains the source code for the Leather Leg Warmers product landing page.

## About
Luxury handmade leather leg warmers designed for style and elegance. As seen on runways, they can be worn over shoes, pumps, booties, or ankle boots to create the look of boots from ankle to knee, regardless of heel height. Visually elongates legs and adds a sexy, high-fashion touch.

Variations have been created by famous designers such as Karl Lagerfeld for Chanel, Phillip Lim, and Givenchy.

## Features
- 100% genuine leather
- Product image and description
- Size selector (S, M, L, XL)
- Stripe & PayPal checkout buttons
- Size chart for reference
- Optimized for GitHub Pages

[https://leatherlegwarmers.com](https://leatherlegwarmers.com)

## Usage
1. Clone or download this repository.
2. Update the `index.html` with your actual Stripe/PayPal links and product images.
3. Deploy to GitHub Pages (main branch) or any static hosting platform.
